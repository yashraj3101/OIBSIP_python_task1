This Python voice assistant program listens to your commands and responds using speech recognition + text-to-speech.

 Voice Command   | Output/Action                 
 
 **your name**   | “my name is peter”            
 **old are you** | “I am two years old”          
 **time**        | Current system time bolta hai 
  **youtube**     | YouTube website open         
 **whatsapp**    | WhatsApp Web open             
 **joke**        | Ek English joke sunaata hai   
 **exit**        | Assistant band ho jata hai
